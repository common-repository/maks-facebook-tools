== Description ==
This simple plugin add a like/share button before post content and replace default comment box for Facebook comments.
 
*under development

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/maks-facebook-tools` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
